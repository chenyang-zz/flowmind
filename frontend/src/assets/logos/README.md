# FlowMind Logo 设计

## 极简高级版

### 设计原则
- **Less is More** - 去除一切非必要元素
- **精准比例** - 黄金分割和对齐
- **大量留白** - 让视觉呼吸
- **单一焦点** - 一个核心元素

---

## 方案对比

### 1️⃣ 神经网络极简 ⭐ **推荐**
**文件**: `logo-neural.svg`

```
中心大圆 + 四周小圆 + 细线连接
```

**特点**:
- 去掉动画，静态展示
- 减少节点（5个核心点）
- 超细连接线（opacity 0.2）
- 大量留白

**适用**: 主 logo、应用图标

---

### 2️⃣ 字母极简
**文件**: `logo-typography.svg`

```
FM 字母抽象化
```

**特点**:
- 纯线条构成
- 8px 统一宽度
- 方角线端（square cap）
- 无填充

**适用**: 文档、印刷

---

### 3️⃣ 单点焦点
**文件**: `logo-minimal.svg`

```
中心圆 + 两圈光晕
```

**特点**:
- 最极简方案
- 渐进 opacity
- 超细线条（0.5px）
- 禅意感

**适用**: favicon、小图标

---

### 4️⃣ 流动线条
**文件**: `logo-wave.svg`

```
三条渐变曲线
```

**特点**:
- 抽象波形
- 渐变色
- 层次 opacity
- 动态感（静态）

**适用**: loading、背景

---

### 5️⃣ 几何三角
**文件**: `logo-geometric.svg`

```
正三角 + 倒三角 + 中心点
```

**特点**:
- 神圣几何
- 稳定感
- 层叠关系
- 中心聚焦

**适用**: 技术、安全场景

---

## 使用建议

### 主 Logo
**推荐**: `logo-neural.svg`（神经网络极简）
- 识别度高
- 寓意清晰
- 高级感强

### 菜单栏图标
**使用**: `icon-32.svg` / `icon-16.svg`
- 简化版本
- 保持核心元素

### Favicon
**推荐**: `logo-minimal.svg`（单点焦点）
- 16x16 清晰
- 视觉聚焦

### 印刷/单色
**推荐**: `logo-typography.svg`（字母极简）
- 纯线条
- 无需颜色

## 设计细节

### 颜色
```css
主色: #6366f1 (Indigo 500)
辅色: #818cf8 (Indigo 400)
背景: #09090b (Near black)
```

### 线条
```css
粗线条: 8px (字母)
细线条: 1.5px (连接)
超细线: 0.5px (光晕)
```

### 圆角
```css
外框: 56px (应用图标)
内圆: 2-8px (元素)
```

---

## 导出建议

### PNG 尺寸
```bash
256x256 - 应用图标
128x128 - Retina 菜单栏
64x64 - 标准菜单栏
32x32 - 小图标
16x16 - Favicon
```

### 格式
- SVG: 矢量源文件
- PNG: 位图导出
- ICNS: macOS 应用
- ICO: Windows 应用
