# 常见问题 (FAQ)

---

## 通用问题

### FlowMind 是什么？

FlowMind 是一个 AI 驱动的工作流智能体，通过监控你的操作、发现重复模式、自动生成脚本，帮助你提升工作效率。

### FlowMind 支持哪些平台？

目前支持 macOS 13.0+。未来计划支持 Windows 和 Linux。

### FlowMind 是开源的吗？

是的，FlowMind 采用 MIT 开源协议。

## 安装和设置

### 如何安装 FlowMind？

详见 [开发环境搭建](./implementation/01-development-setup.md)。

### 需要哪些权限？

FlowMind 需要以下 macOS 权限：
- 辅助功能（监听操作）
- 完全磁盘访问（文件操作）
- 通知（推送提醒）

### 如何获取 Claude API Key？

访问 [Anthropic Console](https://console.anthropic.com/) 创建 API Key。

## 功能使用

### 如何创建自动化？

1. 打开自动化编辑器
2. 描述你的需求（自然语言）
3. AI 生成脚本
4. 确认并保存

### 模式如何被识别？

FlowMind 使用 PrefixSpan 算法分析事件序列，自动发现重复操作模式。

### 知识库如何工作？

1. 剪藏内容（快捷键或浏览器扩展）
2. AI 自动分类、打标签、生成摘要
3. 向量化存储
4. 语义搜索推荐

## 隐私和安全

### 我的数据安全吗？

是的，所有数据存储在本地，不上传到云端（除 AI API 调用外）。

### FlowMind 会记录密码吗？

不会。敏感信息过滤器会自动排除密码、API 密钥等。

### 可以删除数据吗？

可以。设置中有数据删除选项，也可以删除特定类型的数据。

## 性能问题

### FlowMind 占用多少资源？

目标：后台运行时 < 2% CPU，< 200MB 内存。

### 如何降低资源占用？

1. 关闭不需要的监控（如剪贴板）
2. 调整事件保留时间
3. 减少模式挖掘频率

---

## 故障排除

### 应用无法启动

1. 检查权限是否授予
2. 查看日志文件
3. 尝试重启应用

### 监控不工作

1. 检查辅助功能权限
2. 确认监控类型已启用
3. 查看日志排查错误

### AI 不响应

1. 检查网络连接
2. 验证 API Key 是否正确
3. 查看 Ollama 是否运行（本地模式）
